# TAVÉINE — Project Rules

Design system and coding standards for the entire project. Every decision should reinforce luxury, emotion, and restraint.

---

## Brand

TAVÉINE is not a flower shop.

It is a luxury storytelling brand.

Every interaction should feel emotional, cinematic, and minimal.

**Inspired by:**

- Apple
- A24
- Saint Laurent
- Nothing Technology

---

## Colors

| Role | Value |
|------|-------|
| Background | `#000000` |
| Primary text | `#FFFFFF` |
| Secondary text | `#A1A1AA` |
| Accent | `#D4AF37` |

Never use bright colors.

---

## Typography

- Elegant
- Thin
- Large spacing
- Minimal
- No rounded playful fonts

---

## Animation

- Very slow
- Very smooth
- No bouncing
- Fade
- Blur
- Opacity
- Scale

Everything must feel premium.

---

## Layout

- Lots of empty space
- Centered content
- One message per screen
- No clutter

---

## Code Rules

- Always write clean production-ready code
- Split components when needed
- Use Tailwind CSS
- Use Framer Motion
- Avoid unnecessary libraries
- Comment important sections
- Never generate low-quality UI
- Everything should look premium
